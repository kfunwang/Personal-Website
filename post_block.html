行內元素與區塊元素的差別

Photo by Khadeeja Yasser on Unsplash

前言

前一陣子製作自己的個人網站時，發現自己有一些基礎觀念沒有真的搞懂，其中一個就是行內元素與區塊元素的差別 ! 每一個HTML元素都有預設的顯示值，了解元素的預設值除了能幫助你更快速的切版，也能避免不必要的設定，讓Code更乾淨 !

如果你現在也對行內元素及區塊元素感到困惑，歡迎繼續往下看 ~

什麼是行內元素?

  • 預設值 : display : inline
  • 在HTML當中會和前後內容放置在同一行，直到一行排不下才會換行，寬度隨元素的内容而變化
  • 常見 : a、img、i、br、input、spane、code

特性

  • 設定width、height無效，只能由內容撑起来
  • 對margin僅設定左右方向有效，上下無效；padding設定上下左右都有效
  • 行內元素中不能再包含區塊元素

什麼是區塊元素?

  • 預設值 : display : block
  • 在HTML當中會獨自占掉一行的空間 ( 向左右延伸直到填滿父元素寬度 )
  • 常見 : h1 ~ h6、p、ul、ol、li、div、main、aside、article、section、form

特性

  • 設定width、height、padding、margin均有效

inline、block二合一 : inline-block

如果想讓元素並列，同時又能設定寬高，那就是inline-block使用的最佳時機了 !

  • display : inline-block
  • 常用於擴大a的點擊範圍
